## 과제 수행 과정 요약 

1. 초기 파일 생성 및 푸시
* `a.txt` 파일을 생성하고 커밋했습니다.
* Push 시 Git 충돌(non-fast-forward)이 발생하여 Pull-Merge 과정을 거친 후 푸시했습니다.

2. 최종 PDF 파일 제출
* `sourcetree_capture.pdf` 파일을 생성하고 커밋했습니다.
* SourceTree UI에서 파일이 인식되지 않아 터미널을 사용했습니다.
* 최종적으로 `git push origin main -f` (강제 푸시)명령어를 사용하여 모든 파일을 성공적으로 전송 완료했습니다.# github-hw
